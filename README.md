# soilnircompiler

**Calibration transfer between a low-cost NIR sensor and a research-grade
spectroradiometer for predicting soil properties**

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXXXXX.svg)](https://doi.org/10.5281/zenodo.21506455)

## Overview

This repository contains the R pipeline developed to evaluate whether a
low-cost near-infrared (NIR) sensor can replace a research-grade
spectroradiometer for predicting key soil fertility properties, using
spectral calibration transfer.

Two instruments were compared:

- **ASD FieldSpec** (350–2500 nm) — research-grade reference spectroradiometer
- **NIRVascan NIR-S-G1** (Inno-Spectra Corp., 900–1700 nm) — low-cost portable
  NIR sensor (30 scan repeats, 180 patterns, 11.71 nm resolution, 2.54 ms
  exposure time)

Six spectral standardisation approaches were implemented and benchmarked to
transfer NIRVascan spectra into the ASD spectral space:

1. Direct Standardisation (DS)
2. Calibration transfer via global correction factor (CF global)
3. Band-wise / local correction factor (CF, "Lucky Bay" band-wise)
4. Piecewise Direct Standardisation (PDS)
5. Combined CF global + DS
6. **Regularised Band-wise Regression (RBR)** — a novel index introduced in
   this work

Following standardisation, Partial Least Squares Regression (PLSR) and Cubist
models were calibrated to predict:

- Total Carbon (%)
- Total Nitrogen (%)
- pH (H₂O)
- Plant-available Phosphorus (mg/100 g, Double Lactate extraction)

Model performance is reported using MEC (adjusted R²), CCC, RMSE, bias, RPD,
and RPIQ, following standard soil spectroscopy validation practice.

## Soil analysis methodology (reference/wet-chemistry data)

Reference soil property values used to calibrate and validate the spectral
models were generated by conventional wet-chemistry analysis at the
**ZALF Central Laboratory (Zentrallabor), Müncheberg, Germany**:

- **pH** — measured in water suspension (soil:solution ratio per ZALF
  standard protocol; KCl and CaCl₂ extractions also available at the lab but
  water-extractable pH was used here)
- **Total Carbon (Ct) and Total Nitrogen (Nt)** — dry combustion elemental
  analysis
- **Plant-available Phosphorus** — Double Lactate (DL) extraction, a
  standard German agronomic soil-testing method (CAL extraction is also
  offered by the lab as an alternative)

See `metadata/soil_analysis_methodology.md` for a full description of the
analytical methods used (potentiometric pH in water, dry combustion for
Ct/Nt, and Double Lactate extraction for plant-available P).

## Data availability

Raw spectra and reference soil data are archived on Zenodo (see DOI badge
above) rather than committed to this repository. See `data/README.md` for
file descriptions and instructions on where to place them locally before
running the pipeline.

## Requirements

R (≥ 4.2) with: `dplyr`, `tidyr`, `readr`, `stringr`, `tibble`, `prospectr`,
`pls`, `Cubist`, `caret`, `clhs`, `e1071`, `cowplot`, `gtable`, `ggplot2`,
`patchwork`, `parallel`, `doParallel`, `purrr`, `ggdist`, `ggridges`.

The script installs any missing packages automatically on first run.

## Usage

1. Clone this repository.
2. Download the archived dataset from Zenodo and place the CSV files in
   `data/` (see `data/README.md`).
3. Open `scripts/01_calibration_pipeline_v3.R` in RStudio and run — the
   script sets its own working directory relative to the project root and
   creates all required output folders automatically.
